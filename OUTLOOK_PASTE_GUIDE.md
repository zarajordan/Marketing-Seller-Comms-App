# How to Paste HTML Directly into IBM Outlook

## Quick Start Guide

### Method 1: Copy HTML to Clipboard (Recommended) ⭐

This is the **easiest and fastest** way to get your formatted email into Outlook!

#### Steps:
1. **Create your email** in the IBM Marketing Comms App
2. Click the **"📋 Copy HTML for Outlook"** button (primary blue button)
3. Wait for the success message: "✅ HTML copied!"
4. **Open IBM Outlook** and create a new email
5. Click in the email body
6. **Paste** using `Ctrl + V` (Windows) or `Cmd + V` (Mac)
7. Your formatted email will appear with all styling intact! 🎉

#### What Gets Copied:
- ✅ Banner/hero image
- ✅ All text formatting (bold, italic, underline, etc.)
- ✅ Colors and fonts
- ✅ CTA buttons
- ✅ Links
- ✅ All styling

### Method 2: Export to Outlook (.eml file)

If copy-paste doesn't work, use this alternative:

#### Steps:
1. Click **"Export to Outlook (.eml)"** button
2. Your default email client will open with the content
3. Alternatively, download the .eml file
4. Double-click the .eml file to open in Outlook
5. Forward or edit as needed

## Tips for Best Results

### ✅ Do's
- **Test first**: Send yourself a test email to verify formatting
- **Check images**: Ensure banner images are properly sized (600px × 90-200px)
- **Preview**: Use "Preview Email" button before copying

### ❌ Don'ts
- Don't paste into plain text mode in Outlook
- Don't use extremely large images (keep under 1MB)
- Don't forget to add recipients after pasting

## Troubleshooting

### Problem: Formatting doesn't appear when pasted
**Solution**: 
- Make sure you're pasting into the email body, not subject line
- Ensure Outlook is in HTML mode (not plain text)
- Try Method 2 (.eml export) instead

### Problem: Images don't show
**Solution**:
- Images are embedded as base64 data URLs
- Some email clients may block external images
- Recipients may need to "Show Images" in their email client
- For best results, use images from the gallery (already optimized)

### Problem: Copy button doesn't work
**Solution**:
- Your browser may not support the Clipboard API
- Try using "Export as HTML" and open the file
- Copy the content from the HTML file and paste into Outlook

### Problem: Colors look different in Outlook
**Solution**:
- Outlook uses Word's rendering engine which may display colors slightly differently
- Test with your actual Outlook version
- Stick to web-safe colors for consistency

## Keyboard Shortcuts

- `Ctrl/Cmd + V`: Paste HTML into Outlook
- `Ctrl/Cmd + C`: Copy (after selecting text)
- `Ctrl/Cmd + S`: Save draft (in app)

## FAQ

**Q: Can I edit the email after pasting into Outlook?**
A: Yes! Once pasted, you can edit text, add more content, or modify formatting directly in Outlook.

**Q: Will recipients see the formatting?**
A: Yes, as long as they're viewing HTML emails (most modern email clients do by default).

**Q: Can I use this for other email clients (Gmail, Apple Mail)?**
A: Yes! The HTML works in any email client that supports HTML emails.

**Q: What if my company blocks HTML emails?**
A: Use "Export as Plain Text" for a text-only version.

**Q: Can I save templates?**
A: Yes! Use "Save as Draft" to save your email for later use.

**Q: How do I include multiple images?**
A: Currently, one banner image is supported. For multiple images, consider using the HTML export and editing manually.
